# Beer Archive — фан‑сайт Ильи Юровского и Ильи Лапинского

Небольшой фан‑проект о пиве, вдохновлённый Ильёй Юровским. Здесь собрана библиотека сортов с оценками по ключевым критериям, а также предусмотрены пользовательские комментарии и собственные оценки. Идея проста: делиться опытом дегустаций и поддерживать живое сообщество любителей пива.

## Описание проекта

Проект — это витрина сортов с подробными карточками, рейтингом и комментариями. Пользователи могут:
- просматривать сортовую библиотеку;
- читать оценки и отзывы;
- оставлять собственные оценки и комментарии.

## На чём сделан

- `Next.js` (App Router)
- `React` + `TypeScript`
- `Tailwind CSS`
- `Framer Motion`

## Как запустить

1. Установить зависимости:
```bash
npm install
```

2. Настроить окружение:
Создай файл `.env.local` на основе `.env.example`:
```
NEXT_PUBLIC_API_BASE_URL=http://localhost:8001
NEXT_PUBLIC_ASSET_BASE_URL=http://localhost:8001
```

`NEXT_PUBLIC_ASSET_BASE_URL` можно не задавать, если статика отдаётся тем же сервером, что и API.

3. Запустить проект:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`.
